🔒 隱私權政策

最後更新日期：2026 年 5 月 16 日
適用應用程式：Conversation Transcription（iOS）
資料管理單位：臺灣集中保管結算所

一、概述

本應用程式（以下稱「本 App」）由臺灣集中保管結算所開發並維護，僅供本所內部員工使用。 本政策說明我們如何蒐集、使用及保護您的個人資料。

二、我們蒐集的資料

透過 Google 帳號登入取得：

  • 您的 Google 帳號姓名與電子郵件地址（用於身分驗證與使用權限確認）
  • 您 Google 日曆中的個人行程事件（僅限讀取，用於顯示會議清單）

使用 App 過程中產生：

  • 語音錄音檔（儲存於您的裝置本機）
  • 語音辨識後的文字轉錄內容（儲存於您的裝置本機）
  • 翻譯後的文字內容（儲存於您的裝置本機）
  • 使用紀錄 log（含帳號名稱、登入時間、功能使用記錄，上傳至 Azure Blob Storage）

三、資料的使用目的

我們蒐集的資料僅用於以下目的：

  1. 驗證使用者身分，確認是否具備使用本 App 的授權
  2. 顯示您的 Google 日曆會議列表，方便建立對應翻譯紀錄
  3. 提供即時語音翻譯、轉錄及 AI 摘要功能
  4. 系統管理與錯誤排查

我們不會將您的資料用於廣告、行銷或出售給任何第三方。

四、Google API 資料使用聲明

本 App 使用 Google API 取得的資料，嚴格遵守 Google API 服務使用者資料政策（Google API Services User Data Policy）， 包含限制使用規定（Limited Use Requirements）。

Use of information received from Google APIs will adhere to the Google API Services User Data Policy , including the Limited Use requirements.

具體而言：

  • 我們僅將 Google 日曆資料用於在本 App 內顯示您的會議列表
  • 我們不會將 Google 日曆資料傳送給任何第三方
  • 我們不會使用 Google 日曆資料訓練 AI 模型
  • 我們不會在提供您直接要求的功能以外，儲存或使用 Google 資料

五、資料儲存位置

  • 語音錄音、轉錄文字、翻譯內容：僅儲存於您的 iOS 裝置本機
  • iCloud 備份：如您啟用，資料會依 Apple 隱私政策備份至 iCloud
  • 使用 log：僅上傳帳號名稱、Email、操作時間至本所管理的 Azure Blob Storage
  • AI 摘要處理：文字內容傳送至本所訂閱的 Azure OpenAI 服務進行處理

所有雲端服務均由臺灣集中保管結算所訂閱並管理，資料不會落入任何商業第三方手中。

六、資料保存期間

  • 裝置本機資料：由使用者自行刪除或解除安裝 App 時移除
  • 使用 log：保存於 Azure Blob Storage，依本所資訊安全政策定期清除

七、您的資料權利

您有權：

  • 查詢本所保存的您的相關 log 資料
  • 要求刪除您的使用 log
  • 撤銷本 App 對您 Google 帳號的存取授權（至 Google 帳戶設定 > 安全性 > 第三方存取 撤銷）

如需行使上述權利，請聯繫：chengyingtw@gmail.com

八、未成年人

本 App 僅供本所員工使用，不適用於 18 歲以下人士。我們不會刻意蒐集未成年人之個人資料。

九、隱私政策變更

本政策如有重大變更，將透過 App 內通知或 Email 告知使用者。 繼續使用本 App 即表示您同意更新後的政策內容。

十、聯絡我們

如對本隱私政策有任何疑問：
